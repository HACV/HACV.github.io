        # 各大『物理机』厂商进入BIOS

        # 一.『ASUS』vivoBook

        在ASUS出来之前，按下『esc』

        然后，自行选定U盘。非常方便，弄掉U盘也方便

        # 二.『ASUS』台式机

        • 可能有F2

        # 三.『联想』小新

        # （1）第1步—设置3板斧

        快捷键**『F2』**

        原因：好像联想的BIOS，新版的OS设置了要一直使用默认硬盘，有安全策略，所以，我们进行设置

        1、Security中 Security Boot从Enabled设置为disable

        2、Boot中，Boot Mode从UFF1设置为Legacy Support

        3、Exit中，OS Optimized Defaults从Enable设置为Disable

        保存，请按照『电脑规定的，不然不会生效』比如上面的这个是按下F10，就能推出保存

        然后系统会自动重启

        # （2）第2次重启后—用『Fn+F12』

        第2次开启，注意，换了快捷键！！

        然后就直接从U盘中引导进去了

        # （3）复原原先的启动项目

        经过前面的设置，我们就算拔掉U盘后，这样启动会让我们去选择media

        为了复原原先安全的模式，我们在启动后，马上用『F2』进入

        然后，修改回3板斧，保存，重启。

        就能复原了
