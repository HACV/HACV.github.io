      # 汇编手动加栈

      # 1）Windows的C++编译器

      C++ 在头文件前面加上这样的编译器指令 演示如下：

      #pragma comment(linker,"/STACK:1024000000,1024000000")

#include<iostream>
using namespace std;

      //1024后面跟上6个0

      # 2）Linux的g++编译器

      g++ 在main函数里面写如下C语言内嵌汇编指令

      int _size=256<<20;	//256MB
char *p=(char *)malloc(_size)+_size;
__asm__
(
"movl %0,%%esp\n"
:
:"r"(p)
);

      注意，asm前后都是两个下划线_(虽然，看起来像前面和后面都只有一根下划线似的）

      汇编开栈不一定适用，和系统有关，需谨慎。
