      # 公司福利

      # 一、十几薪和年终奖

      比如16薪和年终奖

      两者是不同的。

      • 1、单位是否发年终奖，属于单位的自主权利，只要没有写进合同，不发也不违法
      • 2、16薪啥的，只要写进了合同就一定要发，不发是违法

      # 二、5险1金及拓展

      # 『1』五险

      • （1）养老保险

        在你交满15年之后。
等到你退休之后，你每个月是可以拿到养老金的，也是我们常说的『退休金』，每个月会打钱给你
特点：交的时间越长，交的越多，当你退休之后，每个月拿到的钱就会越多

      • （4）失业保险
      拿这个保险的3个要求：

1.你需要在你之前上班的地方，缴纳这个保险1年以上

2.你的这个失业，并不是你自己主动去辞职，而是公司开除你，或者裁员的这个情况。你如果自己主动跳槽，辞职，这个不算

3.你需要在失业的60天里面，去相关的部门，进行登记『这个钱，最多是可以领取2年的』，要是2年之后，那这个钱是不会继续给你发的

PS：『要是你的工作比较稳定：』

比如，事业编、行政编，这些钱算是交的冤枉，因为这个钱是交的比较冤枉的，因为你基本上不会存在失业的问题！

      # 『2』一金：公积金

      • 住房公积金『减少去银行商业贷款，因为银行等贷款会贵一些』
      这个相当有用：

算法：你的工资里面扣多少，你的单位也要给你交多少

这样，我们就知道你是double『双份』
      1
      2
      3
      4
      5
      • 用途：你可以拿来贷款、买房、装修、或者特殊情况，全部拿出来也是可以的，公积金贷款买房，它的利息是非常非常低，可以说是：买房贷款神器
      • 比如你公积金账户上有2000块，那你贷款的额度是：按照你公积金上的钱，乘上一定的倍数，比如说，你可以贷款你公积金账户的20倍，也就是4W块，贷款来的这个利息，你每个月只要慢慢还就好了。只要支付非常小的利息。
      • 然后，买房的话，也需要贷款嘛，也可以直接从你公积金账户上去扣。
      • 公积金有点像：强制性的给你把这比钱存起来，等你要用的时候，再去申请，就可以把这个钱给拿出来。

      注意：五险一金，是从你的工资里面扣的

      • 校招中那些工作岗位，也是算上五险一金之后的总收入，有些企业可能还是给你的税前收入。
      • 因为每个地方和单位，他们五险一金缴纳的比例，都不太一样。一般具体情况，具体去算
