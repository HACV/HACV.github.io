          # 我的写作工具和习惯

          # 一、『Typora』快捷键

          • 插入引用——Ctrl+shift+Q
          • 去除引用——无

          # 二、『Haroopad』

          • 第1-5是Haroopad编辑器的东西

          # （1）重视快捷键，重视工具

          一定要重视工具的使用，对于常用的工具，要观察好自己的使用习惯，积累自己使用的定制方案

          # （2）自己常用的Markdown的语法

          • 粗体 ** ** （三者都支持） Ctrl+B（快捷键）
          • 粗体 <b> </b>（三者都支持）
          • 字体颜色<font color="red"> </font> 字体颜色
          • 字体大小<font size=5> </font>字体大小
          • 字体类型<font face="行书" size=5> </font>字体类型
          • 字体背景颜色<font style="background: red">红色</font><br>字体背景颜色
          • 插入图片和大小<img src="../photos/2020.6.03_15/2020_06_08_1.png" width="80%">
          • 加个换行<br>
          • Markdown一个套兼容HTML的语法，所以使用HTML中下标下标的语法即可。
          • 上标<sup></sup>232
          • 下标<sub></sub>H2O
          • 表格的底色<table><td bgcolor=#FFFF FF></td></table>

          # （3）Haroopad上Markdown常用快捷键

          • ALt+左方向 跳到行最左边
          • ALt+右方向 跳到行最右左边
          • Ctrl+K 将选中的添加行内代码标记
          • Ctrl+Shift+Q 在选中的行前面添加>
          • Ctrl+shift+J 添加行内数学公式 但是3个$$$什么鬼(注意，hexo中的数学公式则只支持一个$)

          # （5）我常用的两种Markdown编辑器对比

          • Typora（目前我常用的-2020.12.30更新）

          官网下载传送门

          （1）优点

          • 1、颜色不错
          • 2、打印之后会自动生成目录！
          • 3、直接可视化，对于笔者熟练Markdown语法后使用很OK
          • 4、有源代码模式和即时显示效果。
          • 5、各种主题打印的版本还不一样，很好看，比起下面的那款
          • 6、打开速度快，节省我准备写这个的时间
          • 7、插入表格比Haroopad快捷

          （2）缺点

          • 数学公式好像没有Haroopad那么OK

          • Haroopad

          官网下载传送门

          优点

          • 数学公式支持比较OK，可视化

          • 1、笔者最先入门的Markdown编辑器，学会了很多快捷键和Markdown基础语法

          • 2、现在可以考虑迁移到Typora,两者同时使用
