        # 一、函数概念辨析

        类的成员函数 类的非成员函数

        静态成员函数static 常量成员函数const 虚函数 友元函数

        • 类的成员函数：是定义为类的一部分的函数，有时也被称为方法。（C++primer第5版，P20）
        • 各类函数中：构造函数、析构函数、拷贝初始化构造函数等是类的成员函数。
