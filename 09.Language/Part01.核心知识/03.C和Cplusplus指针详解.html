    有人的评价不错：

    指针的步长与指针的类型相关，由于p为int型指针，故p+i表示跳过i个int长度；

    指针步长的核心经验

    int aa[5];
//aa是个int * const aaPtr= &aa[0];
//aa是个本身不能变的（也就是指向不能变）的指针，但是指向的对象它本身可变
//『+1』的步长是一个int『4字节』

auto ptr=&aa;
//类型是下面的，数组指针
//『+1』的步长是5个int『20个字节』
int (*ptr)[5]=&aa;
