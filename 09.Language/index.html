                    # 🐾专题4、C++98核心详解

                    工程术语：

面向对象：
    - 1、static
      - 修改变量或函数存储的形式，比如.date，.bss
    - 2、const 
      - 修改变量or函数的只读方式
    - 3、inline
      - 建议在“编译期”，直接将整个函数的代码插入调用语句处
    - 4、friend
      - 声明友元，包括，友元函数，友元类
