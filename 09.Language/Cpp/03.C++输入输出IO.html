      # C++输入输出IO

      # 01.iostream的设计初衷

      • 对初学者友好
      • 提供一个高效的可扩展的类型安全的IO机制

      iostream的现状

      • 在真实的项目中很少用到iostream
        • 参考自“陈硕”

      # 02.iostream的特点

      • 不同于标准库其他class的“值语义(value semantics)”，iostream是“对象语义(object semantics)”
        • 即iostream是non-copyable
        • iostream禁止拷贝，利用对象的生命期来明确管理资源（如文件）
      • iostream在线程安全方面没有保证！
