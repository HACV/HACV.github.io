            Linux read命令用于从标准输入读取数值。

            # （1024）man的经验

            大家都知道在Unix/Linux中有个man命令，可以查询常用的命令，函数。可是对于我们这样只知道用"man 函数名"来查询的人来说，会遇到很多问题，比如：

            man read，我想看的是ANSI C中stdio的read函数原型和说明，没想到出来的确是BASH命令的说明，这是怎么回事呢？

            原来read本身是man命令的一个参数，这样输入man就会以为你要使用read的功能，而不是查看read函数，那么要怎样查看read函数呢?

            要查属于哪一部分的，就用哪一部分的编号在命令之前。

            此外，在Ubuntu中是可以安装中文man的，不会的可以GOOGLE下

            1.man 2 write

            man后加2表示：函数是系统函数。

            2.man 3 fopen

            man后加3表示：函数是C库函数。

            # （10）Linux下开发C语言需要安装的manpages手册『核心』

            • 因为笔者发现，在我买的『腾讯云』的cetos服务器上，我想man fork和man scanf都没有反应！
            • 比如，我man系统编程的man 7 signal也显示
            • No manual entry for signal in section 7
            • 如果知道本电脑装上了哪些man pages呢？
            这个简单 man 全部内容基本上都在这个目录（包括了目录）里。
cd /usr/share/man
            1
            2

            linux下开发C代码需要安装的manpages：

            $ sudo apt-get install manpages
$ sudo apt-get install manpages-dev
$ sudo apt-get install manpages-posix
$ sudo apt-get install manpages-posix-dev

manpages 包含 GNU/Linux 的基本操作
manpages-dev 包含 GNU/Linux 的基本操作API
manpages-posix 包含 POSIX 所定义公用程序的方法
manpages-posix-dev 包含 POSIX 的 header files 和 library calls 的用法
