          # Windows10『电脑经验』

          # 一、小黑记事本

          win10中txt图标变成小黑记事本的图标——怎么改回来

          1.首先要卸载掉小黑记事本（相关卸载比较麻烦）
2.删掉后，图标还是小黑记事本的，很可恶
3.将txt右击，打开方式改为写字板
4.然后将txt右击，打开方式改为记事本就OK了

          # 二、让谷歌浏览器一直用flash的注册表

          • 将下面内容保存为注册表文件，test.reg，执行
          Windows Registry Editor Version 5.00  

[HKEY_CURRENT_USER\SOFTWARE\Policies\Chromium] 

"AllowOutdatedPlugins"=dword:00000001 

"RunAllFlashInAllowMode"=dword:00000001 

"DefaultPluginsSetting"=dword:00000001 

"HardwareAccelerationModeEnabled"=dword:00000001 

[HKEY_CURRENT_USER\SOFTWARE\Policies\Chromium\PluginsAllowedForUrls] 

"1"="https://*" 

"2"="http://*" 

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Chrome] 

"AllowOutdatedPlugins"=dword:00000001 

"RunAllFlashInAllowMode"=dword:00000001 

"DefaultPluginsSetting"=dword:00000001 

"HardwareAccelerationModeEnabled"=dword:00000001 

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Chrome\PluginsAllowedForUrls] 

"1"="https://*" 

"2"="http://*"

          # 三、Windows10中小娜助手无法搜索本地应用怎么办

          1.在开始—所有应用中找到“Windows PowerShell”；
2、或者在cortana搜索框搜索PowerShell，右键以管理员身份运行；
然后执行下面的命令
          1
          2
          3

          命令：

          Get-AppXPackage -Name Microsoft.Windows.Cortana | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
